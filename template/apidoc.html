API Documentation

State API

Return Different State details ---
State Name, State Code, Source from where hospital data is taken

GET: /state

Base URL

https://fightcovidindia.herokuapp.com/api

District API

Return Different district details ---
Districte Name, District Code

GET : /state/statecode

GET : /state/27

Returns a list of district data including District Name, District Code

Details API

Return Different What Details and What Bedtype are Available about hospitals of a state

For Example For State of Bihar Bedtypes are Total and ICU and For Uttarakhand General, Oxygen, ICU and Ventilator are Available

GET : /details/statecode

Example

GET : /details/5

Output
{
'bedtype': 
   ['Total', 'ICU'],
'details': 
   [
      'name',
      'updatetime', 
      'Contact', 
      'type', 
      'category'
    ]
        }
Returns a type of bed and details available about Hospital of a state

Hospital API

GET : hospitals/statecode/districtcode

Near Hospital API

Return Hospital within provided Distance from provided Location (lat, lon)

GET : /nearhospitals/statecode/lat/lon/distance
Example
GET : /nearhospitals/5/24.88726/85.54591/1
Output

Filter hospital by available Bed Type

Return Hospital within district with available Bed

/availablebedtype/statecode/districtcode/bedtype
Example
First Get Available bed type of a State by using Details Api
GET : /details/27
Output
{
'details': 
   [
      'name',
      'type', 
      'updatetime'
   ],
'bedtype':
   [
      'Isolation', 
      'Oxygen', 
      'Ventilator'
   ]
}
Then filter bed with one of the bedtype
GET : /availablebedtype/27/528/Oxygen
